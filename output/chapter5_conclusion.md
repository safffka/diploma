# 5. Заключение

В рамках выпускной квалификационной работы проведено сравнительное исследование семи архитектур нейронных сетей для семантической сегментации аэрофотоснимков на наборе данных ISPRS Vaihingen. Реализованы и обучены модели: FCN, U-Net, DeepLabV3+, Attention U-Net, SegFormer, а также модификации U-Net и Attention U-Net с предобученным энкодером ResNet50.

Основные результаты исследования:

1. Предобученный энкодер ResNet50 обеспечивает наибольший прирост качества сегментации (+6,06 п.п. mIoU по сравнению с обучением с нуля), превосходя вклад любых архитектурных модификаций декодера. Лучшая модель — U-Net с предобученным ResNet50 — достигает mIoU 61,82% на тестовой выборке.

2. Механизмы внимания (Attention Gate, Multi-Head Self-Attention) не обеспечивают статистически значимого улучшения mIoU (−1,38 п.п. при обучении с нуля, −0,39 п.п. при использовании предобученного энкодера), однако улучшают качество делинеации границ на +0,63–0,72 п.п. по метрике Boundary IoU.

3. Вычислительные накладные расходы механизмов внимания минимальны (1–10% дополнительных FLOPs), что подтверждает гипотезу H2.

4. SegFormer демонстрирует наилучшую вычислительную эффективность (2,3 млн параметров, 2,9 GFLOPs) при конкурентоспособном качестве, сопоставимом с DeepLabV3+.

**Практические рекомендации.** При наличии ограниченного объёма аннотированных данных дистанционного зондирования рекомендуется использовать архитектуру U-Net с предобученным на ImageNet энкодером. Добавление механизмов внимания целесообразно в задачах, где критично качество границ (например, кадастровая съёмка), при этом вычислительные затраты увеличиваются незначительно. Для ресурсоограниченных сценариев (бортовая обработка, мобильные устройства) перспективен SegFormer.

**Ограничения исследования.** Работа имеет ряд ограничений, которые следует учитывать при интерпретации результатов: (1) обучение выполнено на 16 аннотированных изображениях одного города, что ограничивает обобщаемость выводов; (2) аппаратные ограничения (Apple MPS без CUDA) обусловили минимальный размер мини-пакета (batch_size=2), что может влиять на стабильность BatchNorm-статистик; (3) не исследованы альтернативные стратегии аугментации и планирования скорости обучения; (4) не проведена кросс-валидация из-за вычислительных ограничений.

**Научная новизна.** Получены количественные оценки влияния механизмов внимания на качество семантической сегментации аэрофотоснимков в условиях малого объёма обучающих данных. Показано, что при 220 тренировочных тайлах вклад предобученного энкодера превышает вклад архитектурных модификаций в 4–15 раз, что имеет практическое значение для выбора стратегии разработки систем анализа данных дистанционного зондирования.

**Направления дальнейших исследований.** Перспективными направлениями являются: (1) масштабирование на наборы данных большего объёма (Potsdam, LoveDA) для проверки воспроизводимости результатов; (2) исследование эффективности механизмов внимания при увеличении объёма обучающих данных — гипотетически, на выборках свыше 1000 тайлов эффект может стать статистически значимым; (3) применение полусупервизированного обучения для использования неаннотированных изображений; (4) апробация современных предобученных энкодеров (ConvNeXt, Swin Transformer) в качестве альтернативы ResNet50; (5) оптимизация моделей для развёртывания на бортовых системах через дистилляцию знаний и квантование.
